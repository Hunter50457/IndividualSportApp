# IndividualSportApp

**IndividualSportApp** — это индивидуальное спортивное приложение для Android, разработанное на Java в Android Studio.

## Figma

[Figma](https://www.figma.com/design/TXaM3uruJoMeolUal1hBVy/IndividualSportApp?node-id=1-2&p=f&t=Ch2ZtEPCAHzJQoYf-0)

## Описание проекта

Приложение помогает пользователю следить за тренировками, выполнять упражнения по плану, считать индекс массы тела и контролировать ежедневную активность.

Проект разработан для Android 13 с использованием API 33.

## Основные возможности

- Индивидуальная настройка имени пользователя
- Настройка дневной цели по тренировкам
- Отслеживание прогресса тренировки за день
- Таймер тренировки на 20 минут
- Калькулятор ИМТ
- Просмотр плана тренировки
- Возможность изменять план тренировки
- Сохранение данных пользователя через SharedPreferences
- Сброс плана тренировки к стандартному варианту

## Технологии

- Java
- Android Studio
- Android SDK API 33
- XML / Java UI
- SharedPreferences
- Gradle

## Требования

- Android Studio
- Android SDK 33
- JDK 17 или выше
- Android 6.0+ для запуска приложения
